# LukzFilmes

Um projeto Java para gerenciamento de filmes.

## Estrutura do Projeto

- **Filme.java** - Classe que representa um filme com informações como nome, duração, gênero, avaliação, etc.
- **Main.java** - Classe principal da aplicação

## Características

- Classe `Filme` com atributos:
  - Nome do filme
  - Duração em minutos
  - Gênero
  - Inclusão no plano de assinatura
  - Avaliação
  - Total de avaliações

## Como Usar

1. Clone o repositório
2. Abra o projeto em sua IDE Java favorita (IntelliJ IDEA recomendado)
3. Execute a classe Main

## Requisitos

- Java 8+
- IDE Java (IntelliJ IDEA, Eclipse, etc.)
